title：上传图片
date：2019-11-07 11：06：28

tags：

1.打开博客配置文件_config.yml，将post_asset_folder这个选项设置为true
2.在Hexo安装目录下执行:npm install hexo-asset-image –save，这是下载安装一个可以上传本地图片的插件
3.等待一小段时间后，再运行hexo n “xxxx”来生成md博文时，/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹BP1
4.最后在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，BP2然后只需要在xxxx.md中按照markdown的格式引入图片：BP3 注意，此处的图片路径必须使用相对路径
5。执行hexo g,检查生成的页面中图片的src地址。此时生成页面中图片src地址应该与页面的相对路径一致（具体路径取决于页面路径格式设置
